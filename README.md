# Semaphore CLI

## Installation

Before installation:

``` bash
mkdir -p ~/.sst 

echo "<api_token>" > ~/.sst/api_token
echo "<api_domain>" > ~/.sst/api_domain
```

``` bash
mkdir -p ~/bin

wget "" -O ~/bin/semaphore

chmod +x ~/bin/semaphore
```

Optionally append the following line to your `~/.bash_rc` or `~/.zshrc`:

``` bash
export PATH="$HOME/bin:$PATH"
```
