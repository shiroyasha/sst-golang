## Installation

The following command will install Semaphore CLI to the `~/bin` directory.

``` bash
curl -L "https://goo.gl/brnQfs" | sh
```

Optionally append the following line to your `~/.bash_rc` or `~/.zshrc`:

``` bash
export PATH="$HOME/bin:$PATH"
```
